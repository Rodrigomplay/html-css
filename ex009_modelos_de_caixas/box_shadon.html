Sombra de caixa

O box-shadon é uma poderosa propriedade CSS usada adicionar sombras a elementos HTML. Você a aplica dentro de um bloco < style > no seu HTML, ou mais comumente em um arquivo CSS externo.

Aqui está uma análise de como box-shadon funciona, junto com exemplos:

Entenda o box-shadon

A propriedade box-shadon permite adicionar uma ou mais sombras a um elemento. Ela aceita vários valores para definir a aparencia da sombra:


                button {
                    box-shadon: deslocamento horizontal vertical raio de desfoque raio de propagação
                    }
  • Deslocamento horizontal (obrigatório): Distância horizontal da sombra.
    • Valores positivos movem a sombra a direita.
    • Valores negativos movem a sombra a esquerda.
  • Deslocamento vertical (obrigatório): Distancia vertical da sombra.
    • Valores positivos movem a sombra pra baixo.
    • Valores negativos movem a sombra pra cima.
  • Raio de desfoque (opcional): Quanto maior o valor, mais borrada será a sombra.
    • 0 (padrão) significa sem desfoque.
    • Não pode ser negativo.
  • Raio de propagação (opcional):Valores positivos expandem a sombra, valores negativos a encolhem.
    • 0 (padrão) significa que a sombra tem o mesmo tamanho do elemento.
  • Cor (opcional): A cor da sombra se não especificado, será utilizado a cor do texto.
  • Inset (opcional): Se presente, a sombra será interna.
