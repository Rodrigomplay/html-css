Modelo de caixas

Um conceito fundamental para organizar e estilizar o conteúdo na web! Existem dois tipo de caixas que o css utiliza para rendenizar os elementos. Caixas de nível de bloco (block-level boxes) e caixas de nível em linha (inline-level boxes)

1. Caixas de nível de bloco(Block-Level Boxes):

Imagine essas caixas como blocos sólidos que ocupam toda a largura disponível do seu elemento pai (a não ser que você defina uma largura específica). Por padrão, os elementos de nível de bloco sempre começam em uma nova linha.

Caracteristicas principais:

  • Ocupam toda a largura disponível: Tentam se estender para preencher a largura do seu conteiner pai.
  • Forçam quebra de linha: Elementos de nível de bloco antes e depois deles aparecem em linhas separadas.
  • Respeitam as propriedades de largura (width) e altura (height):Você pode definir explicitamente a largura e a altura dessas caixas.
  • Respeitam as propriedades de margem(margin) e preenchimento(padding) em todas as quatro direções (topo,inferior,direita e esquerda)

Exemplo de elementos HTML que geram caixas de nível de bloco por padrão:

  • < div >
  • < p >
  • < h1 > a < h6 >
  • < ul >
  • < ol >
  • < li >
  • < form >
  • < form >
  • < header >
  • < footer >
  • < section >

Exemplo em CSS:


            div {
                background-color: lightblue;
                width:50%;
                padding:20px;
                margin-botton:15px;
            }

Neste exemplo, cada div ocupara 50% da largura do seu conteiner, terá um preenchimento interno de 20 pixels e uma margem inferior de 15 pixels, aparecendo um abaixo do outro.

Caixas em nível de linha (inline-level boxes)

Pense nessas caixas como palavras dentro de uma frase. Elas ocupam apenas o espaço necessário para exibir seu conteúdo e não forçam quebras de linhas

Caracteristicas principais:

  • Ocupam apenas a largura necessária para o conteúdo: A largura da caixa se ajusta ao conteúdo dentro dela.
  • Não forçam quebras de linha: Elementos de nível em linha podem aparecer lado a lado na mesma linha.
  • Ignoram as propriedades de largura(width) e altura (height): Tentar definir width e height em elementos inline geralmente não tem efeito ( há exceções com inline-block).
  • Respeitam as propriedades de margem (margin) e preenchimento(padding) apenas nas direções horizontal (esquerda e direita): As margens e o preenchimento vertical podem afetar a renderização de elementos adjacentes de maneira inesperada.

    • Exemplos de elementos HTML que geram caixas de nível em linha por padrão:

    • < span >
    • < a >
    • < img >
    • < strong >
    • < em >
    • < br >
    • < input >
    • < select >
    • < textarea >
    • < code >

    Exemplo em CSS:

    
            span {
                background-color: blue;
                padding: 5px 10px;
                margin: 0 5px;
            }

    Neste exemplo, cada < span > terá um fundo azul, um preenchimento de 5 pixels em cima e embaixo ,10 pixels nas laterais e uma margem de 5 pixels nas laterais, aparecendo lado a lado se houver espaço. Observe que a margem vertical pode não afetar o espaçamento vertical de outros elementos inline da mesma forma que a margem horizontal.

    O poder do display no CSS:

    A grande sacada é que você pode alterar o tipo de caixa de um elemento usando o display. Os valores mais comuns são:

    • display: block; Força um elemento a sem comportar como uma caixa de nivel de bloco.
    • display:inline; Força um elemento a se comportar como uma caixa de nível de bloco.
    • display: inline-block; Uma combinação poderosa! O elemento flui como um elemento em linha (não força quebra de linha), mas respeita as propriedades de largura, altura, margem e preenchimento em todas as direções como um elemento de bloco.
    • display:flex e display:grid; Criam contextos de layout flexivel e de grade respectivamente, alternando a forma como os filtros diretos do elemento são dispostos.
    • display:none; Remove completamente o elemento do fluxo do documento (ele não ocupa nenhum espaço e não é exibido).

    Podemos utilizar a ferramenta devtools. Em elementos você pode visualizar o box padrão ou configurado por você.É possível ajustar, mas a alteração não salva, precisa copiar e colar.

    • height = altura
    • width = largura
    • border = borda da caixa
    • padding = Espaço que distancia a borda do conteúdo
    • margin = parte de fora da caixa
    • outline = contorno da borda
