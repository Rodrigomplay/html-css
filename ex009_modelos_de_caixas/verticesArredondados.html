Border-Radius

O que é border-radius?

Basicamente o border-radius define o raio da curvatura que será aplicada a cada um dos quatro cantos de um elemento. Você pode aplicar o mesmo raio a todos os cantos, ou especificar raios diferentes para cada um, criando formas mais complexas.

Sintaxe e valores:

Existem várias maneiras de aplicar o border-radius, desde a mais simples até a mais especifica:

  1. Arredondamento uniforme (todos o cantos iguais)
    • Se você fornecer um único valor, ele será aplicado a todos os quatro cantos do elemento.
      • 
                HTML
                < div class="caixa arredontada" > </div>

                CSS
                .caixa {
                       width: 150px;
                       heigth: 100px;
                       background-color: #F0F0F0;
                       border: 2px solid #333;
                       margin: 20px;
                       }

                .arredondada {
                       border-radius: 15px; /* Todos os cantos com raio de 15px */
                       }
    • Com dois valores:
      • 
                        border-radius: valor1 valor2;
      • valor 1 é aplicado aos cantos superior esquerdo e inferior direito
      • valor 2 é aplicado aos cantos superior direito e inferior esquerdo
    • Com três valores
      • 
                        border-radius: valor1 valor2 valor3;
      • Valor 1 é apçicado ao canto superior esquerdo.
      • Valor 2 é aplicado aos cantos superior direito e inferior esquerdo.
      • Valor 3 é aplicado ao canto inferior direito.
  2. Criando formas circulares ou elípticas

    3. Usar porcentagem é especialmente útil para criar círculos ou elípses:

    • Círculo perfeito: Para criar um círculo perfeito a partir de um elemento quadrado (com width e height iguais), defina border-radius: 50%;

    HTML

    
                    < div class="circulo" > </div>

    CSS

        
                    .circulo {
                        width: 100px;
                        height:100px;
                        background-color: tomato;
                        border-radius: 50%;
                    }
  • Elipse: Para criar uma elipse a partir de um elemento retângular, você também pode usar 50% ou valores específicos para criar um arredondamento elíptico. Você pode usar a barra / para especificar raios horizontais e verticais independentes para cada canto, criando formas elípticas.

                border-radius: 
                [raio-horizontal-1] [raio-horizontal-2] [raio-horizontal-3]  [raio-horizontal-4] / 
                [raio-vertical-1] [raio-vertical-2] [raio-vertical-3]  [raio-vertical-4];

Ou de forma mais simples, se quiser uma elipse uniforme:

CSS


                    .elipse {
                    width: 200px;
                    height: 100px;
                    background-color: lightgreen;
                    border-radius: 50% / 50%; /* Ou 100px / 50px para esse exemplo */
                    }
