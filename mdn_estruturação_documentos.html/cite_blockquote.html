Tag cite e blockquote

Entendendo a tag < cite >

A tag <cite> é usada para marcar a fonte de uma citação. Pense nela como uma forma de dar crédito à origem do texto que você está citando dentro do seu HTML.

Para que ela serve, em resumo:

Indicar a fonte: Ela informa aos navegadores e outras ferramentas que o conteúdo dentro dela é o título de um trabalho (como um livro, artigo, música, filme, etc.) que foi citado no texto principal.

Semântica: Ela adiciona significado semântico ao seu documento HTML, tornando-o mais compreensível para mecanismos de busca, leitores de tela e outros agentes de usuário. Isso ajuda a entender a estrutura e o contexto do seu conteúdo.

Estilização padrão (geralmente): Por padrão, muitos navegadores renderizam o texto dentro da tag < cite > em itálico. No entanto, essa estilização pode ser alterada com CSS.

Como ela é usada:

A tag <cite> é geralmente usada dentro de um elemento de citação, como <blockquote> (para citações mais longas) ou <q> (para citações curtas em linha).

Exemplo:

1. Usando com <blockquote>:

Como disse Albert Einstein

A imaginação é mais importante que o conhecimento. O conhecimento é limitado, enquanto a imaginação abraça o mundo inteiro,estimulando o progresso, dando luz a evolução.

Albert Einstein

Neste exemplo,< cite > Albert Einsten </cite> indica que a citação acima é de Albert Einsten

Entendendo a tag blockquote

A tag <blockquote> em HTML serve para indicar uma seção de texto que é citada de outra fonte. Pense nela como uma forma de destacar um trecho de conteúdo que não é original do seu documento, mas sim retirado de algum outro lugar.

Para que ela serve, em resumo:

  • Indicar citação longa: Ela sinaliza para os navegadores e outros agentes de usuário que o conteúdo dentro dela é uma citação extensa de outra fonte.
  • Semântica: Assim como a tag <cite>, ela adiciona significado semântico ao seu HTML, deixando claro qual parte do texto é uma citação. Isso melhora a acessibilidade e a interpretação do conteúdo por máquinas e pessoas.
  • Estilização padrão (geralmente): Por padrão, muitos navegadores renderizam o conteúdo dentro de <blockquote> com uma margem à esquerda (e às vezes à direita), criando um recuo visual para diferenciá-lo do texto principal. Essa estilização padrão pode ser modificada com CSS.

Como ela é usada:

A tag <blockquote> envolve o texto que você está citando. Geralmente, dentro dela, você terá um ou mais elementos <p> (parágrafos) contendo a citação. É comum também usar a tag <cite> dentro ou fora de <blockquote> para indicar a fonte da citação.

Exemplos:

Em seu famoso livro, "O Pequeno Príncipe", Antoine de Saint-Exupéry escreveu:

Tu te tornas eternamente responsável por aquilo que cativas.

O Pequeno Príncipe, Antoine de Saint-Exupéry

Como observou o renomado cientista:

A ciência sem religião é manca, a religião sem ciência é cega.

Albert Einstein
