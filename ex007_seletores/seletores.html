Criando site com HTML e CSS

Seletores

Os seletores em HTML sáo denominados como id ou class

Os seletores seletores são denominados por #e .

Exemplo:


        < h1 id="principal"> Criando site com HTML e CSS </h1>

No exemplo acima é uma seleção em HTML. Pra CSS declaramos da seguinte forma:


        h1#principal {
            text-align: center;
            background-color: lightblue;
            color: blue;
        }

Ou podemos declarar somente como #principal

    Pesquisar mais sobre seletores personalizados:
  • id em HTML = # em CSS
  • class em HTML = . em CSS
  • pseudo-class em CSS = :
  • pseudo-elementos em CSS = ::
  • > = children

Recomenda-se utilizar somente um id. Caso queira identificar mais elementos recomenda-se utililaz o class. É possivel colocar uma classe dentro de id. É possivel também colocar mais elementos dentro de class
