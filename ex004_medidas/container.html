Container

O termo container em html não se refere a uma tag especifica com esse nome (embora possamos usar <div> ou <section> como containers).

Ele se refere a idéia de agrupar elementos HTML relacionados dentro de um elemento pai. Esse agrupamento serve para dois propósitos principais:

  1. Organização estrutural: Tornar o cõdigo HTML mais semântico e fácil de entender, dividindo o conteúdo em seções lógicas. Por exemplo, você pode ter um container para:
    • Cabeçalho(<header>)
    • Navegação(<nav>)
    • Conteúdo principal(<main>)
    • Barra lateral (<aside>)
    • Rodapé (<footer>)

    Você pode ter outros containers(<section>) ou (<div>) para agrupar artigos,formulários,etc.

  2. Estilização com CSS: Facilitar a aplicação de estilos a um grupo de elementos de uma vez. Ao envolver um conjunto de elementos em um container,

Tags HTML comumente usadas como containers:

  • <div>(Divisão): É um container genérico, sem nenum significado semântico inerente. É usado puramente para agrupar outros elementos para fins de estilo ou script. 
    • 
                <div class="container-principal">
                    <h1>Título da seção</h1>
                    <p>Conteúdo da seção</p>
                        <ul>
                          <li>Item</li>
                          <li>Item</li>
                        </ul>
                </div>
  • <section>(Seção): Representa uma seção temática de conteúdo relacionado em um documento.É mais semântico que <div> e ajuda a definir a estrutura do documento.
    • 
            <section id="sobre-nos">
                <h2>Sobre nós</h2>
                <p>Informações sobre a nossa empresa...</p>
            </section>
  • <article>(Artigo): Representa uma composição autocontida em um documento, página, aplicativo ou site.Pode ser um post de blog, um artigo de jornal, etc.
    • 
                <article>
                    <h3>Título do artigo</h3>
                    <p>Conteúdo do artigo...<p>
                </article>
  • <header>(Cabeçalho): Geralmente contém informações introdutórias ou um conjunto de links de navegação.
  • <nav>(Navegação): Usado para agrupar links de navegação
  • <aside>(Conteúdo lateral): Representa conteúdo relacionado ao conteúdo principal, mas que pode ser considerado separado(como uma barra lateral).
  • <footer>(Rodapé): Geralmente contém informações sobre o autor,direitos autorais, links para documentos relacionados, etc.
