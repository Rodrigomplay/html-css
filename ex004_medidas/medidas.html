Medidas relativas e absolutas

As medidas são representadas de duas formas: (medidas relativas e absolutas)

Medidas absolutas

As medidas absolutas representam um valor fixo e não se alteram dependendo de outros fatores, como o tamanho da tela, a fonte do usuário ou o elemento pai.Elas correspondem a unidades de medida físicas ou ponto fixos.

As unidades absolutas mais comum em CSS são:

  • px (pixels): Um pixel é a menor unidade de medida na tela de um dispositivo. É uma unidade fixa e previsivel
    • Obs:Embora seja absoluta, a percepção do tamanho em pixels pode variar ligeiramente entre telas com diferentes densidades de pixels (DPI ou PPI)
  • cm (centímetros): Uma unidade de medida física.1cm corresponde a um centímetro real.
    • Obs: O uso de cm, mm, in, pt e pc é menos comum para exibição na tela,pois o tamanho real pode variar dependendo do dispositivo de saída (monitor, impressora, etc).Elas são mais utilizadas em estilos para impressão.
  • mm (milímetros): Similar a cm,mas em milímetros.10mm equivale a 1 cm.
  • in (polegadas):Outra unidade de medida física. 1 in equivale a 2.54cm.
  • pt (pontos):Uma unidade tradicionalmente usada em tipografia.1 pt equivale a 1/72 de uma polegada
  • pc (picas). Outra unidade tipografica. 1pc equivale a 12 pt.

Quando usar medidas absolutas:

  • Quando você precisa de um tamanho fixo e que não se altere, independentemente do contexto.
  • Em estilos para impressão, onde as dimensões físicas são importantes.
  • Para bordas(geralmente px) onde uma expessura consistente é desejada.

Desvantagens das medidas absolutas para web:

  • Pouca flexibilidade:Elementos com tamanhos absolutos podem não se adaptar bem a diferentes tamanhos de tela ou resoluções, podendo causar barras de rolagem indesejadas em dispositivos menores.
  • Problemas de acessibilidade: Usuários que aumentam o tamanho da fonte padrão do navegador podem ter dificuldades em visualizar o conteúdo se os tamanhos dos elementos estiverem fixos em pixels.

Medidas relativas

As medidas relativas baseiam seus valores em relação a algum outro valor, como o tamanho da fonte do elemento raiz (<html>),a largura da viewport (área visível do navegador) ou a altura da viewport. Isso torna os layouts mais flexiveis e adaptáveis a diferentes contextos.

As unidades relativas mais comuns são:

  • em:(O nome "em"tem origem na tipografia tradicional, onde a largura da letra "M"maiúscula era frequentemente usada como unidade de medida). Relativo ao tamanho da fonte do elemento pai. Se o elemento pai tem uma font-size de 16 px, entçao 1 em no elemento filho também será 16px.Se o elemento filho tiver sua própria fonte-size definida, os valores em em dentro dele serão relativos a essa nova font-size.
    • 
            body {
               font-size: 16px; 
            }
            div {
                font-size: 1.5; /* 1.5 * 16px = 24px */
                padding: 1em; /* 1 * 24px = 24px */
            }
            p {
                font-size: 0.8em; /* 0.8 * 16px = 12.8px */
            }
    Obs: Valores em em podem se acumular. Se um elemento dentro de outro elemento também usar em, o tamanho será relativo ao em do elemento mais próximo.
  • rem (relativo ao tamanho da fonte do elemento raiz): "rem" significa "root em". Diferentemente de em, que é relativo ao pai mais próximo, rem é sempre relativo ao tamanho da fonte do elemento raiz do documento HTML (geralmente o elemento ). Isso torna o rem mais previsível para dimensionar elementos de forma consistente em todo o site, especialmente para tipografia.
    • 
            html {
                font-size:16px;
            }
             div {
                font-size:1.5rem; /* 1.5 * 16px = 24px */
                padding: 1rem; /* 1 * 16px */
             }
             p {
                font-size: 0.8rem; /* 0.8 * 16px = 12.8px */
             }
  • vw (relativo à largura da viewport): "vw" significa "viewport width". 1 vw é igual a 1% da largura da janela do navegador (a viewport). Por exemplo, se a largura da janela for 1200px, então 1 vw será 12px, e um elemento com width: 50vw terá 600px de largura.
  • vh (relativo à altura da viewport): "vh" significa "viewport height". Similar ao vw, 1 vh é igual a 1% da altura da janela do navegador (a viewport). Se a altura da janela for 800px, então 1 vh será 8px, e um elemento com height: 100vh ocupará toda a altura da tela.
  • % (porcentagem, relativo ao elemento pai): A unidade de porcentagem é relativa a uma propriedade do elemento pai. Para propriedades como width, height, margin, e padding, a porcentagem é geralmente relativa à dimensão correspondente do elemento pai. Por exemplo, se um elemento filho tem width: 50% e seu pai tem uma largura de 400px, o filho terá uma largura de 200px. Para a propriedade font-size, a porcentagem é relativa ao tamanho da fonte do elemento pai, funcionando de forma semelhante ao em.
