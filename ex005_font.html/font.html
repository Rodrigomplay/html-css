Trabalhando com fontes em CSS

Medidas absolutas:

cm, mm, in, px, pt, pc

Medidas relativas:

em, ex, rem, vw, vh, %

A recomendação da w3scholls é que se utilize px ou em. A medida rem é utilizada no root. As medidas absolutas sáo mais utilizadas para o caso de impressáo.

shorthand font:

Em h1 se utiliza muito esse tipo de configuração:


        h1 {
            font-family: "work Sans", sans-serif; /*Define o tipo de letra*/
            font-weight: bolder; /*Define a espessura (ou peso) da fonte*/
            font-size: 3em; /*Define o tamanho da fonte*/
            font-style: italic; /*Define o estilo da fonte (No caso italic)*/
           }

Podemos utilizar o método shorthand que fica exatamente nessa ordem (font-style => font-weight => font-size=> font-family)

 h1 { font: italic bolder 3em 'work Sans', sans-serif;}

Capturando fontes de um site

Para isso podemos instalar uma extensão em chrome web store chamada fonts ninja. Essa extensão tem a versão paga, mas podemos utilizar somente para identificar a fonte.

Como utilizar?

  • Instalar a extensão fonts ninja em chrome web store
  • Abrir o site que você quer identificar a fonte
  • Clicar no icone da extensão fonts ninja

Detectando fontes dentro de imagens

whatfontis.com ou font finder, font squirrel, myfonts.com

Alinhamento de texto

  • text-align:justify; Fica alinhado dos dois lados.
  • text-align:left; fica alinhado a esquerda
  • text-align: right; Fica alinhado a direita
  • text-align: center; Fica centralizado
  • text-indent: 30px; O inicio do paragrafo fica um pouco a direita.

Seletores personalizados

Podemos selecionar se um h1,h2 fica uma a direita e outro a esquerda. Para isso precisamos identicar por classe. Exemplo < h2 class 1 >
